R Code for using R-INLA to implement the multinomial logit modelling analysis described in:

"Hancock, P.A., Lynd, A., Wiebe, A., Devine, M., Essandoh, J., Wat'senga, F., Manzambi, E., Agossa, F., Donnelly, M.J., Weetman, D., Moyes, C.L., 2022, Modelling spatiotemporal trends in the frequency of genetic mutations conferring insecticide target-site resistance in African malaria vector species", BMC Biology"

This script requires the following the input following files:
1. "multinomial inla.r": File containing R commands for processing input data and running the INLA multinomial logit regression model
2. "load_gen.r": File containing functions needed to run the script
3. "stk_val_inds_kdr_wa_ea_sp3.r": indices of data points to withhold for the 10 test sets
3. "nn_oos_preds3.RData": The 10-fold out-of-sample predictions generated by the neural network model
4. "rf_oos_preds3.RData": The 10-fold out-of-sample predictions generated by the random forest model
5. "xgb_oos_preds3.RData":  The 10-fold out-of-sample predictions generated by the extreme gradient boosting model
6. "inputs_data_all_wa_ea_sp3.r": Data set with all the sample Vgsc allele frequencies and environmental predictor variables
